Simple BEM selector engine

## Installation

```bash
npm install bemquery-selector-engine --save-dev
```

## BEM selectors syntax

`block` → `.block`
`block elem` → `.block__elem`
`block:modifer` → `.block_modifier`
`block > other-block` → `.block .other-block`
